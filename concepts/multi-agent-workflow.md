# Multi-Agent 워크플로우

## 개요
4가지 역할의 AI 에이전트가 협력하는 자동화 시스템

## 4단계 구조

### 1️⃣ Planner Agent (기획)
- 레이아웃 설계
- 색 팔레트 정의
- 섹션 구성 계획

### 2️⃣ Designer Agent (디자인)
- [[agent-reach]]로 정보 수집
- [[Codex]]로 이미지 생성
- 색감 일관성 유지

### 3️⃣ Developer Agent (개발)
- HTML/CSS/JavaScript 작성
- 반응형 디자인 구현
- 애니메이션 추가

### 4️⃣ Reviewer Agent (검토)
- QA 보고서 생성
- 색감 검증
- 반응형 확인
